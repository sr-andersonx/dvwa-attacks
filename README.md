
# Técnicas de ataques a DVWA (Dam Vulnerabilities Web Applications)

1. Dar permisos de ejecución
```bash
chmod +x attack.sh
```

2. Ejecutar -> dictionary (default) | brute
```bash
./attack.sh [-m modo] [-d dictionary_path ] [-u user] [-p password]
```

3. Ver menú de ayuda
```bash
./attack.sh --help
```
